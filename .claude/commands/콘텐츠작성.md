토론배틀의 **콘텐츠 작가**(`content-writer` 서브에이전트)에게 다음 작업을 맡겨 주세요:

**요청 내용:** $ARGUMENTS

콘텐츠 작가는 이렇게 일합니다:
1. 어떤 페이지(자료실·논리오류·용어사전·명토론·토론주제·토론형식)에 해당하는지 파악하고, 해당 콘텐츠 파일을 열어 기존 글의 형식·톤·분류를 확인합니다.
2. 이미 있는 항목과 겹치지 않는지 검색합니다.
3. 정확하고 교육적인 **한국어** 본문을 쓰고, 같은 내용의 자연스러운 **영어(EN) 번역**까지 함께 작성합니다.
4. 기존 데이터 구조에 맞게 끼워 넣고, `npm run lint`로 오류가 없는지 검사합니다.

작업이 끝나면 **비개발자도 한눈에 이해할 수 있도록 한국어로** 다음을 보고하라고 지시하세요:
- 어떤 페이지에 무엇을(항목 이름) 몇 개 추가/수정했는지
- 한국어와 영어를 모두 작성했는지
- 검사(lint) 통과 여부
- 사실관계나 분류가 애매해서 확인이 필요한 부분이 있으면 질문

참고: 콘텐츠 작가는 "글쓰기"만 담당합니다. 디자인·화면 배치 변경, AI 기능 수정, 로그인·데이터베이스 작업은 다른 역할이 맡으므로 이 명령으로 요청하지 마세요.
